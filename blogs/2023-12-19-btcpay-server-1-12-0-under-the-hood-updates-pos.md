BTCPay Server 1.12.0 brings under-the-hood updates, point-of-sale improvements, and BoltCards issuance at scale. With PostgreSQL 11 reaching end-of-life, BTCPay Server no longer supports it and recommends upgrading to a more recent PostgreSQL version.
