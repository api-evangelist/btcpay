BTCPay Server 2.2.0 adds enhanced reporting, a public plugin builder, Miniscript support, an LSP plugin, and GiveWP integration. New wallet report columns show transaction fee and exchange rate, exchange rates are now captured automatically when a transaction is detected, and stores can track additional currencies such as USD or GBP alongside their default for multi-currency accounting.
