A case study on how Strainly, a freemium web app for hemp growers and breeders, used BTCPay Server for peer-to-peer and private payments at scale. Setup on a dedicated VPS took about an hour, and Strainly used the API and webhooks to automatically upgrade members to paid tiers once a BTCPay invoice is settled, enabling Lightning payments from the start to minimize fees.
