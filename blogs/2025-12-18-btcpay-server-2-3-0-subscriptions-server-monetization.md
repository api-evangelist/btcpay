BTCPay Server 2.3.0 introduces subscriptions, server monetization, a plugin builder, and Drupal 11 and EDD integrations. Because Bitcoin operates as a push mechanism requiring authorization rather than the pull model of credit cards, subscriptions are designed to be explicit and transparent while still offering customers a smooth renewal experience.
